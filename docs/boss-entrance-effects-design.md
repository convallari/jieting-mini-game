# 《街亭小游戏》Boss 出场特效设计

## 1. 设计目标

Boss 出场要在 2～3 秒内同时完成四件事：让玩家立刻认出角色、看懂威胁类型、明确 Boss 从单一路线入口出现，并产生一次值得期待的视觉高潮。

核心原则：

- **先预警，再登场**：危险信息必须早于 Boss 实体出现，避免无提示扣血。
- **每名 Boss 一种视觉语法**：张郃是赤红军阵，魏军先锋是青蓝断水，司马懿是靛黑天幕。
- **演出不偷时间**：演出期间波次计时和敌人移动暂停；玩家可以观察，但不可误触棋盘。
- **单路聚焦**：所有引导线、镜头运动和粒子都指向现有左路入口，不再出现第二入口暗示。
- **可跳过但不突兀**：重复观看时点击或按空格可压缩到 0.35 秒收尾段。

## 2. 通用出场流程

总时长建议为 `2.4s`，分为五段：

| 阶段 | 时间 | 画面与交互 |
| --- | ---: | --- |
| 危险预兆 | 0.00–0.45s | 环境降亮，入口出现角色主题色脉冲；普通敌人停步并向两侧避让。 |
| 名号落印 | 0.45–0.95s | 巨型 Boss 名号以毛笔扫写出现，随后像印章一样砸向入口。 |
| 本体破场 | 0.95–1.45s | Boss 从墨雾、军旗或水浪中现身；镜头轻震一次。 |
| 威胁展示 | 1.45–2.05s | 用短暂的非伤害预演展示其技能范围或战术目标。 |
| 回到战斗 | 2.05–2.40s | 名号缩入 Boss 血条，遮罩退场，移动和波次计时恢复。 |

通用 UI：

- 顶部临时 Boss 条显示姓名、技能图标和一句威胁提示。
- 首次遇见显示技能提示；重复遇见只显示姓名与技能名。
- Boss 条从屏幕上方压下，结束后固定在波次信息下方。
- 出场期间保留营寨生命和军粮数值可见，不遮挡关键资源。

## 3. 张郃——「赤阵压山」

定位：第一位正式主将，强调纪律、压迫和封锁阵型。主题色为朱红、铁灰、暗金。

### 分镜

1. **军鼓逼近（0.00–0.45s）**  
   战场亮度降至 70%，左路入口沿路径连续亮起三枚赤色脚印。每枚脚印伴随一声由远及近的鼓点。
2. **军令斩落（0.45–0.90s）**  
   一道横向朱红笔锋扫过画面，写出“张郃”。名字末笔变成一面魏军令旗，重重插在入口格。
3. **列阵现身（0.90–1.45s）**  
   六个半透明“魏”字残影在入口两侧列阵，向中央合拢；张郃从中央铁灰墨雾中踏出。合拢瞬间产生短促屏幕震动与尘环。
4. **压迫预演（1.45–2.05s）**  
   以张郃为圆心展开一圈红色方阵格线，短暂扫过可能被其“压迫”影响的我方单位。单位脚下只出现锁形预警，不实际施加状态。
5. **收尾（2.05–2.40s）**  
   残影化为暗红碎墨，被吸入 Boss 血条；提示语为“阵脚受压 · 合成将被封锁”。

### 好玩点

- 鼓点和路径脚印准确告诉玩家 Boss 从哪里来。
- 预演真实技能范围，使出场动画本身承担教学功能。
- 张郃每次再次出场时，列阵残影数量可随难度增加，但时长不增加。

## 4. 魏军先锋「锋」——「裂水开道」

定位：没有姓名的精锐先锋，强调高速、破坏水源和直线冲锋。主题色为青蓝、冷白、黑铁。

### 分镜

1. **水脉逆流（0.00–0.40s）**  
   地图水源格由营寨方向向入口方向逆向闪烁，水声突然抽空；所有水格表面出现细小裂纹。
2. **锋字破幕（0.40–0.85s）**  
   一个巨大冷白“锋”字由入口向下高速划过，像刀锋切开宣纸。裂口后方露出深蓝水流与骑兵剪影。
3. **踏浪入场（0.85–1.35s）**  
   先锋沿路线冲入半格后急停，身后水浪撞成扇形墨花。急停产生横向残影和一次更尖锐、幅度较小的震屏。
4. **断水预演（1.35–1.95s）**  
   一条青蓝色虚线从 Boss 连到水源格，再延伸至军粮图标；水源格依次变暗，军粮图标短暂抖动，但数值不减少。
5. **收尾（1.95–2.25s）**  
   纸面裂口被黑墨缝合，留下细窄刀痕；提示语为“汲道告急 · 守住水源”。

### 好玩点

- 让地图上的水源参与演出，而不是只在入口播放粒子。
- “锋”字既是角色名，也是斩开画面的武器。
- 先锋血量低于 50% 时，入口刀痕可再次微亮，作为第二阶段提示。

## 5. 司马懿——「玄幕落星」

定位：终局统帅，强调全局控制、深不可测和号令全军。主题色为靛蓝、玄黑、苍白。

### 分镜

1. **万籁骤静（0.00–0.50s）**  
   战场音效快速低通，只剩风声。画面四角涌入靛黑墨云，中央棋盘仍保持可见。
2. **棋局显形（0.50–1.05s）**  
   棋盘上方浮现一层半透明星盘，三枚苍白棋子依次落下：入口、水源、营寨。落子声替代军鼓，最后一子落在营寨上方但不造成伤害。
3. **影中现身（1.05–1.55s）**  
   左路入口先出现司马懿的长影，影子逆向延伸到画面中央，本体随后从影子中升起。名字“司马懿”不是直接写出，而由星盘连线拼合。
4. **号令预演（1.55–2.20s）**  
   从司马懿向场上所有敌军发出靛蓝命令波纹；普通敌人短暂出现加速残影并齐声前倾，但仍保持暂停。全图边缘闪过一次苍白电光。
5. **收尾（2.20–2.65s）**  
   星盘快速旋转并收进 Boss 血条的技能图标；环境音恢复，司马懿轻笑一声。提示语为“全军受令 · 敌军即将强化”。

### 好玩点

- 先落“入口—水源—营寨”三子，直接表达司马懿掌控整张地图。
- 其他敌军参与号令动作，让终局 Boss 的统帅身份成立。
- 若司马懿与张郃同场，只播放司马懿完整演出；张郃以 0.45 秒令旗落地的副将演出加入，避免连续遮屏。

## 6. 交互与镜头规则

- 出场开始时将当前拖拽单位安全放回原格，不能吞掉单位或道具。
- 完整演出仅在该 Boss 本局首次出现时播放；复活、召唤或调试重复生成使用精简版。
- 精简版保留“主题预兆 + 本体破场 + Boss 条”，总时长 `0.65s`。
- 屏幕震动不超过两次：破场 `3px/0.16s`，技能预演 `1.5px/0.12s`。
- 任何单次全屏闪白不超过 80ms，最大亮度不高于背景的 1.35 倍。
- 开启“减少动态效果”时取消缩放、快速残影和震屏，改用淡入、描边与音效提示。

## 7. 音效设计

| Boss | 主声音 | 辅助声音 | 结束标识 |
| --- | --- | --- | --- |
| 张郃 | 三段战鼓 | 旗帜破风、甲片碰撞 | 沉重军令落印 |
| 锋 | 水流抽离 | 刀锋裂纸、骑兵急停 | 水滴落入空井 |
| 司马懿 | 低频风声 | 三次棋子落盘、远雷 | 轻笑与号角低鸣 |

声音应复用现有 `boss_entrance` 作为兼容回退；新增资源加载失败时，演出仍能靠画面和文字完整表达。

## 8. 技术落地建议

新增独立的 `bossEntrance` 状态，不把出场动画塞入普通粒子生命周期：

```js
bossEntrance: {
  active: true,
  bossId: enemy.id,
  bossIndex: 0,
  age: 0,
  duration: 2.4,
  phase: "omen",
  skippableAt: 0.45,
  reducedMotion: false
}
```

建议拆分为：

- `startBossEntrance(enemy)`：暂停波次、处理拖拽、建立演出状态。
- `updateBossEntrance(dt)`：推进阶段、触发一次性音效与震动。
- `drawBossEntranceBackdrop()`：环境遮罩、星盘、纸面裂口。
- `drawBossEntranceForeground()`：名号、令旗、引导线、Boss 条。
- `finishBossEntrance()`：恢复计时与移动，清理残留状态。

现有能力可直接复用：`pulseAt()`、`inkSplash()`、`shake()`、`floatText()`、Spine 本体动画、`boss_entrance` 音效以及现有 Boss 主题色。新增视觉对象建议进入专用数组，避免大量粒子影响普通战斗特效上限。

## 9. 性能预算

- 桌面端同时粒子不超过 110 个，移动端不超过 55 个。
- 全屏遮罩使用渐变和少量路径绘制，不创建逐帧离屏大画布。
- 星盘、裂纸和军旗纹理预渲染并缓存；窗口尺寸变化时才重建。
- Spine 出场只播放一套主动画，剪影和残影优先复用同一渲染结果。
- 出场期间暂停非关键战斗粒子更新，可降低峰值帧耗。

## 10. 验收标准

- 不看文字也能从颜色、声音和动作区分三名 Boss。
- 玩家能在 1 秒内确认 Boss 的入口方向和主要威胁目标。
- 演出期间敌人、波次计时和伤害结算完全暂停。
- 演出结束后，Boss 位置、血量和技能冷却与正常生成一致。
- 手机竖屏与电脑横屏均不遮挡营寨生命、军粮和 Boss 本体。
- 连续跳过、暂停、切后台再恢复都不会卡死演出状态。
- 减少动态效果模式下无强闪烁、强震动或高速全屏运动。
