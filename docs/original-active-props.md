# 原版主动战斗道具

依据 `output/reference/original-web/bundle-readable3.js` 中 `Qi / Zi / Ki` 三个道具类及原版 `props/AutoAtlas`。

## 砚台

- 冷却 90 秒。
- 投放后依次播放 `inkstone_2` 至 `inkstone_5`，随后生成墨迹。
- 作用半径为 1.5 个格子；墨迹存在的 5 秒内，范围内及后来进入范围的敌军速度均降低 20%。
- 本项目在单玩家完整地图上允许选择任意格子作为中心，两条路线的敌军均可受影响。

## 陷阱

- 冷却 50 秒，只能放在道路格。
- 待触发使用 `trap_1`；敌军进入该格后切换为 `trap_2`。
- 触发目标停止移动 5 秒，触发后的陷阱图形保留 5 秒再消失。

## 地雷

- 冷却 55 秒，只能放在道路格。
- 使用 `landmine_1` 与土堆遮罩表现埋设过程。
- 敌军进入所在格时触发爆炸并清除附近敌军。

## 单玩家界面适配

- 参考游戏上下玩家各有一套道具栏；本项目只保留一套共享道具栏，覆盖完整地图和上下两条路线。
- 道具栏贴近地图右侧中线，避免重新引入第二玩家区域或第二套冷却状态。
