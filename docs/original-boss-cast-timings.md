# 原版首领施法时序

时长来自各 Spine JSON 的攻击时间轴；生效点优先来自 `bundle-readable3.js` 中首领类的 `Laya.timer.once` 或动画停止回调。首领施法期间停止沿路线移动，攻击动画完整播放，不能复用普通受击的 180ms 窗口。

| 首领 | 动画时长 | 技能生效点 | 原版依据 |
| --- | ---: | ---: | --- |
| 张梁 | 1.667s | 0.500s开始 | 500–1400ms逐步扩大摄魂扫描半径 |
| 张宝 | 2.000s | 起手至动画结束 | 招魂控制器仅捕获施法期间范围内新死亡的普通敌兵，单次上限3；220ms魂链后原位、原进度满血复活 |
| 张角 | 1.000s | 0.500s | `timer.once(500)` 后给范围内敌军加三项5秒增益：体型+20%、临时生命+50%、移速+30% |
| 孙尚香 | 1.000s | 0.500s | `timer.once(500)` 后生成花并触发拆迁事件 |
| 甄宓 | 1.000s | 末帧 | `attackzhen` 停止回调后启动全场雨与 `-0.2` 攻速减益（80%攻速） |
| 貂蝉 | 2.333s | 起手 | 魅惑Buff立即施加，连线视觉按目标数错峰播放 |
| 华雄 | 0.767s | 约0.367s | `attackhx` 中段召唤回调；整段结束后恢复行走 |
| 吕布 | 1.733s | 0.650s | `timer.once(650)` 后施加方天画戟减级/封锁效果 |
| 董卓 | 2.067s + 0.733s | 0.500s | `timer.once(500)` 吞噬，主动画结束后接 `attack2dz` 膨胀 |
| 典韦 | 0.500s（撞击段） | 撞到目标时 | 原版先冲到随机未击倒目标附近，再走贝塞尔撞击并施加击倒 |

| 曹操 | 1.733s | 末帧 | `attackcao` 停止回调才选择最高级单位并封印 |

当前复刻已将冷却、完整施法、效果触发和恢复行走拆为独立状态；默认速度下 Spine 动画与逻辑秒数一一对应。调试参数 `debugTimeScale` 只用于快速检查逻辑状态，不作为视觉时序验收依据。

## 特殊技能循环

- 张梁不会在0.5秒时一次性命中整个范围。0.5–1.4秒每0.1秒扫描一次，扫描半径随施法进度由中心扩展到完整技能范围；每名普通兵在同一次施法中只会被摄魂一次。
- 貂蝉的 `rE` 返回全屏阵地中所有当前最低等级普通兵。复刻会同时施加魅惑，再从貂蝉到各目标依次绘制弧形连线，而不是只随机挑一个。
- 典韦每次从范围内随机选择一名尚未击倒的兵将。一次施法只击倒这个目标；连续多次施法可以累积击倒不同单位。实体先按原版 `3 × 距离` 毫秒接近到目标侧面0.75格，再用约100ms、控制点上抬200px的二次贝塞尔跃击；接近播放 `godian2`，跃击播放 `attackdian`，撞击后用0.3秒平滑回归路线，避免画面瞬移。
