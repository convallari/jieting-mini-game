# 原版“弓”攻击状态机

依据网页版程序包中的 `Ai`（弓兵）类还原：

- 待机播放 `zhan`。
- 进入攻击时，以当前角度为起点，计算“起点—抬高 120px 的控制点—目标”二次曲线的初始切线方向。
- `attack` 动画先播放 0–650ms；骨骼容器同时用 650ms 转向计算出的切线角度。
- 前段停止后才播放 650–1000ms，并在此时创建箭矢；不能在起手帧提前生成弹道或结算伤害。
- 箭矢沿目标二次曲线飞行并随轨迹切线持续旋转。
- 离开攻击状态后，弓的容器用 650ms 转回 0°。

当前复刻按攻击冷却等比缩放上述 65%/35% 两段，以避免升级攻速后截断末段。原版基础片段总长为 1000ms。箭矢位置使用真实二次贝塞尔公式，箭头旋转直接取该曲线在当前参数处的一阶导数，因此末帧会自然朝向目标，而不是使用正弦抛物线的近似朝向。
