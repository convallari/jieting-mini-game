# 原版敌军数值表

从网页版敌军配置类 `k` 和 `Ct.wy/vy` 提取：

- 普通敌人20级基础生命：`[10,16,26,41,61,92,138,200,291,421,611,886,1285,1863,2701,3917,5680,8235,11941,17315]`。
- 对应每侧出怪次数：`[10,11,12,13,15,16,18,19,21,24,26,29,31,35,38,42,46,51,56,61]`。每1.5秒同时向左右路线各生成一名，所以普通敌人总数为表值的两倍。
- 四类普通敌人的基础生命表相同，基础移动速度均为50像素/秒。
- 三档首领基础生命倍率为7、10、14，基础移动速度均为10像素/秒。
- 波次之间等待5秒。首领只在第3/6/9/12/15/18波判定，出现概率依次为10%/20%/30%/50%/90%/100%；出现时左右路线各进入一名。
- 原版另有局内难度、周目与排行榜加成；当前阶段先还原基础表，附加倍率随波次系统继续接入。

## 首领 Spine 形态绑定

- 每档首领资源内含三套形态，行走和攻击必须使用同一个显式形态索引，不能用占位字的字符编码临时取模。
- `boss0`：`gobao/attackbao`、`gojiao/attackjiao`、`goliang/attackliang`。
- `boss1`：`godiao/attackdiao`、`goxiang/attackxiang`、`gozhen/attackzhen`。
- `boss2`：`gocao/attackcao`、`godian/attackdian`、`goxia/attackdun`。
- 同一波上下两路生成的首领使用同一形态，但两路均归单玩家防守。
