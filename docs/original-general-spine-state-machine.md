# 原版将领 Spine 状态机与迁移规则

依据 `output/reference/original-web/bundle-readable2.js` 中将领基类的实际调用：

- 每名将领同时持有两个同源 Spine 实例：左字 `SU`、右字 `bU`。
- 两个实例分别位于双格容器的左右半区，不是单实例轮播两个字。
- `GeneralIdle`：左字循环播放 `zhan1`，右字循环播放 `zhan2`。
- `UnitAttack`：左字播放一次 `attack1`，右字同时播放一次 `attack2`。
- 范围内持续有目标时保持 `UnitAttack`；每个攻击周期同步重播左右攻击，周期之间保持非循环动画末帧。
- 只有目标列表为空时才返回 `GeneralIdle`，不能在每次攻击结束后自动切回静态动画。
- 攻击播放速率统一由将领攻速 `Fk` 控制。
- 攻击重播间隔为 `1000 × Ok` 毫秒；运行时按当前冷却周期缩放 `attack1/attack2`，不使用独立固定动画时长。
- `shouye` 是主界面赵云专用演出，不属于普通战斗攻击序列。
- `attack4` 不在普通攻击状态转换中，不得与 `attack1/attack2` 顺序轮播。
- 游戏等级 1–3 使用皮肤 `1`，等级 4 使用皮肤 `2`，等级 5 使用皮肤 `3`。
- 合成后的将领视觉和占位均为横向双格；索敌和投射物起点取双格中心。
- 将领攻速累计倍率为 `[1, 1.3, 1.56, 1.794, 1.9734]`，攻击倍率为 `[1, 1.5, 2.1, 2.73, 3.276]`。
- 赵云基础攻击2、范围2.5、间隔0.8秒，优先路线进度最大目标并执行贯穿攻击。
- 张飞基础攻击10、范围2.5、间隔1秒，选择最近目标；动画起手后按 `100 / Fk` ms 延迟执行范围攻击。
- 马超基础攻击10、范围2.5、间隔1秒，选择最近目标并执行单体攻击。

以上仅记录原游戏行为。街亭版本不再把左右字作为两个独立状态机：棋盘保存逐格单字，姓名连续时建立唯一阵型状态机；旧 Spine 两个 part 接收同一语义状态、同一 clip 和同一进度。`attack2/zhan2`保留为旧源素材但默认标记为未映射，不再自动分配给第二字。占格数等于姓名成员数，诸葛亮为三格。
