# 原版普通武器动画时序

从网页版 `Ti.RS` 四个兵种类和序列帧注册表提取：

- 基础数值：刀为攻击3/范围1.5；弓为攻击2/范围3.5；枪为攻击2/范围2.5；骑为攻击2/范围2。四者基础攻击间隔均为0.8秒。
- 1–5级攻击与攻速累计倍率均为 `[1, 1.5, 2.1, 2.73, 3.4125]`，攻击间隔等于 `0.8 / 倍率`。
- 刀、枪、骑选择几何距离最近的目标；弓选择范围内路线进度最大、最接近终点的目标。

- 刀：攻击 19 帧 / 600ms；攻击开始时立即创建贴身命中区域，`500 / Fk` ms 是区域存活窗口，不是延迟命中时间。
- 弓：攻击 30 帧 / 1000ms；0–650ms 瞄准旋转，650–1000ms 放箭。
- 枪：正式骨骼攻击使用21帧；独立枪杆工具依次执行90ms转向、270ms蓄势、120ms前刺并启用命中、90ms收枪，全部按 `Fk` 缩放。审阅台保留的24帧裁片额外包含录屏中的回稳观察帧，不能直接替代正式运行帧数。
- 骑：攻击 19 帧 / 600ms；150ms 后同时启动两道范围扫击，扫击本体持续 300ms。两道命中区域分别覆盖半攻击距离和完整攻击距离，并从 60° 旋转到 -300°；每个区域可依次命中圈内多个敌人，各造成一半攻击力。

所有分段在复刻中按实际攻击冷却等比缩放，避免升级攻速时截断关键命中帧。

原版普通命中特效寿命：刀约 104ms、弓 136ms、枪 256ms、骑 250ms。刀花随攻击方向旋转；普通枪击和弓击不旋转；骑击在攻击方向基础上额外旋转45°。2026-07运行态审计发现旧实现误加135°，现已修正为45°。

固定攻击场景的命中统计证明骑的两个扫击区不是单目标替代物：一次连续审阅中骑产生27次有效命中，而同场刀、枪在被摄魂前各命中1次；每一道扫击对每个进入扇面/圆周的敌人只结算一次，两道扫击各造成一半攻击力，同一敌人可分别被内外圈命中。
